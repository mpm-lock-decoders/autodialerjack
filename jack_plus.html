Autodialers JACK are advanced autodialers that can open most mechanical combination safe locks, including most locks with protection against manipulation. The JACK Plus model is an upgraded version that replaces our award-winning basic JACK model.

All the features of the older JACK

JACK Plus, of course, includes all the great features of the base model, including strong switchable magnets and centering rings for quick installation, a strong base made of lightweight materials, 6 speed settings, adjustable dialing step, optional password protection, ability to re-lock the safe after the combination has been found, and much more.

New control unit, motor and motor driver

The Plus model has a brand new control unit with a new OLED display. The unit has an even stronger high-quality German motor with a better driver.

Improved base

The base now allows a better view of the dial and the control unit can be tilted. The mounting rods are in a square configuration so control the unit can be turned 90 degrees when needed.

Fully upgradeable to the Pro version

Unlike the old JACK model, JACK Plus is fully upgradeable to the Pro version. For a full upgrade, you need to send the unit to us. We will install the missing hardware modules including an SD card reader and a GSM module for SMS notifications and update the software to the Pro version adding the Smart Mode capability for automated dialing and the Diagnostic Mode for locks where the known combination stopped working.

Optional accessories:

Dialing Modes:

  • All combinations (with a variable dialing step)
  • Ranges of combinations specified for each dial separately (with a variable dialing step)
  • Frequently used combinations containing multiples of 5 and 10
  • Common factory default combinations
  • Testing all single number combinations
  • Dialing around a known combination
